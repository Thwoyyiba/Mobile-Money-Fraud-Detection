# Mobile-Money-Fraud-Detection
A Machine Learning based Fraud Detection System using PaySim dataset with Flask deployment to classify mobile money transactions as fraudulent or legitimate.
# Mobile Money Fraud Detection System

## Project Aim

To develop a machine learning based fraud detection system that identifies suspicious mobile money transactions using the PaySim dataset.

## Technologies Used

- Python
- Flask
- Machine Learning
- Scikit-learn
- Decision Tree Algorithm

## Dataset

PaySim Synthetic Mobile Money Transaction Dataset

## Features Used

- step
- type
- amount
- oldbalanceOrg
- newbalanceOrig
- oldbalanceDest
- newbalanceDest
- Acct type
- Date of transaction
- Time of day
- branch
- unusual login
